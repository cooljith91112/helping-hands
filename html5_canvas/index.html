mFragin's HTML5 Canvas Tutorials

This is a collection of HTML5 pages that were originally written for the AETERNUM game project at Skyline High School, started by ThePikminKing. The goal was to make AETERNUM using the Model View Controller design pattern, and one of the implementations of the view and controllers would be the HTML5 Canvas element.

By no means to these form a complete tutorial to the Canvas element. Instead, each individual sketch attempts to show one concept that will be necessary to implement the game in HTML5, on the Canvas. These were created as necessary while I was thinking of what we would be wanting to do for the game.
